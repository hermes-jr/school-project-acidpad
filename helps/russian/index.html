Acidpad

Что такое Acidpad


  Acidpad 1.0 - это простейший текстовый редактор, все функции которого позаимствованы из редакторов Macromedia Homesite и Microsoft Notepad.
  Особенность Acidpad заключается в его переносимости. Он работает одинаково на всех платформах, где установлена виртуальная машина Java (Windows 95/98, Windows 2000/Me/Xp, Linux, Unix, BSD, Sun Solaris и других). Acidpad опробирован и корректно работает в таких ОС, как Microsoft Windows 2000, Windows 98, Windows Me, Windows XP, Mandrake Linux 9.0. Так же Acidpad должен работать под Sun Solaris, но не был опробирован из-за отсутствия программного и соответствующего аппаратного обеспечения.
  Из-за возможных трудностей в установке JVM и самой программы, основнойцелью её создания становится не создание нового (и далеко не первого) простого текстового редактора, а создание отличного и полностью работоспособного примера для начинающих программистов и профессионалов, работающих с Java.
